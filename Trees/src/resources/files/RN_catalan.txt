Un arbre roig-negre é un árbre binari de cerca on cada node té un atribut de color 
roig o negre. A més dels requisits imposotos pels arbres binaris de cerca convencionals,
s'han de cumplir els següents requisits per tindre un arbre roig-negre vàlid:

- Tot node ha de ser roig o negre.
- L'arrell és negra.
- Totes les fulles són negres.
- Tots els fills dels nodes roigs són negres (també anomenada "Propietat del roig").
- Cada camí que uneix un node amb una fulla descendent, conté el mateix nombre de nodes
  negres, o bé contant els nodes nulls, o no contant-los (el resultat és el mateix).
  També es diu "Propietat del camí", i al nombre de nodes negres de cada camí, que és 
  constant per a tots els camís, s'els anomena "Alçada negra del arbre", i per tant el 
  camí no pot tindre dos nodes roig consecutius.
- El camí més llarg des de l'arrel fins una fulla, no és més llarg que 2 vegades el 
  camí més curt des de l'arrel de l'arbre a una fulla d'aquest arbre. El resultat és que
  aquest arbre està aproximadament balancejat.
