If we perform the operation to insert a K KEY with VALUE V inside a tree, tree is 
inserted into the couple <clave, valued> corerspondiente insertion.

POSSIBILITIES:

* If the key K isn't in the tree, the tree will be introduced in the couple <clave, valued>

* If the key K already exists in the tree, proceed to update its value.

* If the key K has an undefined value will generate an ERROR.
